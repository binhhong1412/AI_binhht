# Cho em điểm + nha thầy Trí ^^

## Trạng thái:

Input:

Nhập vào giới hạn của bình 1: a

Nhập vào giới hạn của bình 2: b

Nhập vào mục tiêu: target.

Output:

List các bước giải ra bài toán cho đến đích.

Ví dụ:

In 1: 3

In 2: 4

Target: 1

---

## Trạng thái đầu:

0 0

## Trạng thái cuối:

0 1 hoặc 1 0

## Phép chuyển trạng thái

0 0

0 4

3 1

0 1
